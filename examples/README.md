# Example Files

This directory contains example code files that you can use to test CodeSage.

## Files

- `example.py` - Python example with classes, functions, and type hints
- `example.js` - JavaScript example with classes, async functions, and ES6 features

## Usage

1. Start the CodeSage application (see main README for setup instructions)
2. Navigate to the "Analyze Code" tab
3. Upload one of these example files
4. Explore the analysis results

These examples demonstrate:
- Function definitions
- Class structures
- Type annotations (Python)
- Documentation strings
- Import statements
- Various code patterns

Feel free to modify these files or create your own examples to test different code patterns!
